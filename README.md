Sistema de Gerenciamento de Consultas Médicas

O sistema funciona como uma agenda eletrônica feito para um consultório médico, gerenciando dados que são relacionados aos pacientes, médicos e consultas da clínica através de uma interface simples e organizada



O projeto utiliza a linguagem Java com padrão MVC, tem acesso ao banco de dados MYSQL via JDBC e utiliza Java Swing para criação de interface
